# hideNumber 脱敏

>此组件主要用于对敏感词汇或者敏感数字加*脱敏使用，例如手机号、身份号等，鼠标移入可显示全部信息

## 基础用法
组件使用number属性传值
<show-code showPath="hideNumber/baseHideNumber">
<baseHideNumber></baseHideNumber>
</show-code>


## hideNumber 属性

| 属性       | 说明              | 类型              | 可选值        | 默认值   |
|----------|-----------------|-----------------|------------|-------|
| number   | 绑定值 | string / number | ---        | ---   |
